Initial release of `create-reactive-agent`. Scaffolds a new Reactive Agents project via `npm create reactive-agent`, `bun create reactive-agent`, or `pnpm create reactive-agent`. Three templates (`minimal`, `with-tools`, `streaming`), four providers (`anthropic`, `openai`, `google`, `ollama`), and four package managers (`bun`, `npm`, `pnpm`, `yarn`). Interactive prompts with zero runtime dependencies; non-interactive flags supported for CI use.
