## Start in 60 Seconds

```bash
bun add reactive-agents
```

```typescript
import { ReactiveAgents } from 'reactive-agents'

const agent = await ReactiveAgents.create()
    .withProvider('anthropic')
    .withReasoning() // ReAct loop: Think → Act → Observe
    .withTools() // Built-in: web-search, file-read, code-execute
    .withObservability()
    .build()

const result = await agent.run('Find the top 3 TypeScript testing frameworks')
console.log(result.output)
```

One package. Composable layers. Enable exactly what you need — skip everything you don't.

## Why Reactive Agents?

<CardGrid stagger>
    <Card title="Type-Safe from End to End" icon="seti:typescript">
        **Zero `any`** in framework code. Every agent, tool, memory entry, and
        LLM call is validated by Effect-TS schemas. Failures are typed tagged
        errors, not exceptions. **5,028 tests** keep every service boundary
        honest on every PR.
    </Card>
    <Card title="Composable Layer Architecture" icon="puzzle">
        **30 packages, 13 capability layers.** Each is an independent Effect
        `Layer` with explicit dependencies. Memory without guardrails? Reasoning
        without cost tracking? Just stream tokens? Pick exactly what you need —
        no hidden coupling, no wasted resources.
    </Card>
    <Card title="Observable Execution Engine" icon="magnifier">
        **12-phase deterministic lifecycle** with `before` / `after` / `on-error`
        hooks per phase. Every phase emits spans, metrics, and EventBus events.
        You see what your agent decided, why, in what order, at what cost — no
        manual instrumentation required.
    </Card>
    <Card title="5 Reasoning Strategies" icon="star">
        **ReAct, Reflexion, Plan-Execute, Tree-of-Thought, Adaptive** — plus
        automatic strategy switching when entropy detects the agent is stuck.
        Register your own strategies. ToT outer-loop early-stop and 8-action
        reactive controller ship out of the box.
    </Card>
    <Card title="Local Models That Actually Work" icon="setting">
        **+80pp accuracy** on Ollama 4B+ vs. naive prompting. The 4-stage
        Healing Pipeline recovers from **86.7% of tool-call errors** — 90%
        cheaper than LLM reprompt. Model-adaptive context tunes prompts and
        compaction per tier. Same code, frontier-to-local.
    </Card>
    <Card title="Frontier-Verified" icon="approve-check">
        **100% pass on `ra-full`** across `claude-sonnet-4-6`, `claude-haiku-4-5`,
        `gpt-4o-mini`, and `gemini-2.5-pro` (Apr 30 2026). Bare LLM only achieves
        85% on the same harness — a measurable lift from the framework, not
        the model.
    </Card>
    <Card title="Great DX" icon="rocket">
        **60 seconds to first agent.** Progressive disclosure — start with 3
        lines, add reasoning, memory, guardrails, and observability as you need
        them. The builder API reads like a sentence. `rax` CLI scaffolds, runs,
        and inspects.
    </Card>
    <Card title="Cortex Local Studio" icon="laptop">

## How It's Different

<CardGrid>
    <Card title="vs. LangChain / LlamaIndex" icon="document">
        Python-first, dynamically typed, monolithic. Reactive Agents is
        **TypeScript-native with zero `any`**, fully modular layers, and
        built-in observability. You see every decision — not just the final
        output. Side-by-side migration guide included.
    </Card>
    <Card title="vs. Vercel AI SDK" icon="document">
        Great for streaming and tool calling, but stops there. Reactive Agents
        adds **5 reasoning strategies**, persistent **4-tier memory**,
        guardrails, verification, cost routing, and a **12-phase execution
        engine** with full observability — same TypeScript ergonomics.
    </Card>
    <Card title="vs. AutoGen / CrewAI" icon="document">
        Multi-agent frameworks without type safety, composable architecture, or
        model-adaptive intelligence. Reactive Agents ships **A2A protocol**
        (JSON-RPC + SSE), dynamic sub-agent spawning, and the **healing
        pipeline that lifts local-model accuracy by +80pp**.
    </Card>
    <Card title="vs. Building From Scratch" icon="document">
        **30 production-ready packages, 5,028 tests, 12-phase engine.** Memory,
        reasoning, tools, A2A, gateway, reactive intelligence, safety, cost,
        identity, orchestration — all composable, all opt-in. Focus on your
        agent's logic, not infrastructure.
    </Card>

## Common Patterns

<Tabs>
  <TabItem label="Streaming">
```typescript
// Token-by-token streaming via AsyncGenerator
for await (const event of agent.runStream("Write a haiku about TypeScript")) {
  if (event._tag === "TextDelta") process.stdout.write(event.text);
  if (event._tag === "IterationProgress") console.log(`Step ${event.iteration}/${event.maxIterations}`);
  if (event._tag === "StreamCompleted") console.log("\nDone!");
}

// One-liner SSE endpoint
Bun.serve({ fetch: (req) => AgentStream.toSSE(agent.runStream("Hello")) });

````
  </TabItem>
  <TabItem label="Chat Sessions">
```typescript
// Multi-turn conversation with memory
const session = agent.session();

await session.chat("What's the capital of France?");
// → "Paris is the capital of France."

await session.chat("What's the population?");
// → "Paris has approximately 2.1 million residents..."
// (remembers context from previous turn)
````

  </TabItem>
  <TabItem label="Persistent Gateway">
```typescript
// Autonomous agent that runs 24/7
const agent = await ReactiveAgents.create()
  .withProvider("anthropic")
  .withReasoning()
  .withTools()
  .withGateway({
    heartbeat: { intervalMs: 3_600_000, policy: "adaptive" },
    crons: [{ schedule: "0 9 * * MON", instruction: "Weekly report" }],
    webhooks: [{ path: "/github", adapter: "github" }],
    policies: { dailyTokenBudget: 50_000 },
  })
  .build();

agent.start(); // Runs forever, Ctrl+C to stop

```
