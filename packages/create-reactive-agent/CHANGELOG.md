## 0.11.0

### Minor Changes

- Initial release. Scaffolds Reactive Agents projects via `npm create reactive-agent`.
  - 3 templates: `minimal`, `with-tools`, `streaming`.
  - 4 providers: `anthropic`, `openai`, `google`, `ollama`.
  - 4 package managers: `bun`, `npm`, `pnpm`, `yarn`.
  - Interactive prompts (zero-dep readline) and non-interactive flags.
